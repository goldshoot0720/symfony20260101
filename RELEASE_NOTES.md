# 鋒兄AI資訊系統 - 版本發布說明

## v1.1.0 (2025-01-01) - 系統簡化與優化版本

### ✨ 新功能
- **管理介面**: 新增 `manage.html` 提供直觀的數據管理介面
- **數據庫工具**: 新增 `cleanup_database.php` 和 `create_required_tables.php`
- **本地測試**: 新增 `test_local_server.bat` 自動化測試腳本

### 🔧 系統優化
- **資料庫簡化**: 僅保留必要的 `food` 和 `subscription` 兩個表
- **API 精簡**: 移除影片和圖片相關端點，專注核心功能
- **性能提升**: 優化查詢邏輯和數據處理流程

### 📝 介面改進
- **響應式設計**: 管理介面支援桌面和行動裝置
- **即時統計**: 儀表板顯示食品和訂閱的到期提醒
- **用戶體驗**: 簡化操作流程，提高易用性

### 🗄️ 資料庫結構
- **food 表**: 使用現有結構 (`todate`, `amount`, `photo` 等欄位)
- **subscription 表**: 使用現有結構 (`nextdate`, `site`, `note` 等欄位)
- **數據保留**: 保持現有數據完整性 (張君雅丸子、kiro pro 等)

### 🎯 核心功能
- **食品管理**: 智能追蹤存貨和到期提醒 (3天、7天、30天)
- **訂閱管理**: 管理服務訂閱和付款提醒 (3天、7天內)
- **統計分析**: 即時數據統計和狀態監控
- **API 架構**: RESTful 設計，支援擴展

### 🚀 部署改進
- **本地測試**: 一鍵啟動和測試腳本
- **環境配置**: 簡化配置流程
- **錯誤處理**: 改進錯誤提示和診斷

## 升級說明

### 從 v1.0.0 升級到 v1.1.0
1. 備份現有數據
2. 執行 `php public_html/config/cleanup_database.php` 清理不必要的表
3. 執行 `php public_html/config/create_required_tables.php` 確保必要表存在
4. 測試系統功能

### 注意事項
- v1.1.0 移除了影片和圖片管理功能
- 如需這些功能，請繼續使用 v1.0.0
- 資料庫結構已簡化，專注於食品和訂閱管理
